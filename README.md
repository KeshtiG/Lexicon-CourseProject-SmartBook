# SmartBook – Mini Library Application

This application was developed as the final project for the C# course, part of the
.NET/Fullstack Developer program at Lexicon 2025.

The program is a simple mini library system where users can add, remove, search for,
and sort books. It also includes functionality to load a library from a JSON file
and save the current library to a JSON file.

---

**Usage:**

- Open the solution in Visual Studio using the file Lexicon-CourseProject-SmartBook.sln.
- Run the program by pressing CTRL + F5 or using the Start menu.

---

**Tested Features:**

- Add a book to the library
- Search for a book by title
- Remove a book from the library
